# Healthcare Dashboard - Power BI Project

## 📌 Project Overview
This project is a Healthcare Analytics Dashboard developed using Microsoft Excel and Power BI. The dashboard provides detailed insights into hospital operations, patient management, treatment outcomes, revenue analysis, departmental performance, and patient satisfaction.

The goal of this project is to help healthcare organizations make data-driven decisions by analyzing key operational and patient-related metrics through interactive visualizations.

---

## 🚀 Features

- Interactive KPI Dashboard
- Hospital Operations Analysis
- Patient Demographics Analysis
- Department Performance Monitoring
- ER Waiting Time Analysis
- Revenue Analysis by Age Group
- Bed Occupancy Tracking
- Patient Satisfaction Analysis
- Treatment Outcome Insights

---

  - Data Cleaning

  - Dashboard Development

## 📊 Key KPIs

- Total Patients: 2,506
- Admitted Patients: 1,679
- Outpatients: 827
- Occupied Beds: 1,751
- Total Revenue: $0.404M
- Average Treatment Cost: $161.41
- Average ER Time: 65.63 Minutes
- ICU Patients: 82
- Average Length of Stay: 1.70 Days

## 📈 Dashboard Insights

### Hospital Operations
- Tracks inpatient and outpatient flow
- Monitors patient-to-staff ratio
- Measures emergency response efficiency

### Patient Analysis
- Patient distribution across age groups
- Gender-wise patient demographics
- ICU and discharge statistics

### Department Performance
- Average ER waiting time by department
- Operational efficiency comparison

### Revenue Analytics
- Revenue contribution by patient age group
- Financial performance monitoring

### Patient Satisfaction
- Treatment explanation feedback analysis
- Patient experience tracking

## 🔄 Project Workflow

1. Raw Healthcare Data Collection

4. Data Modeling & KPI Creation
5. Dashboard Development
6. Insight Generation

---

## 📌 Key Findings

- 67% of total patients were admitted as inpatients
- Majority of patients belong to the 6–40 years age group
- High patient satisfaction regarding doctor communication
- OPD department had the highest ER waiting time
- Hospital bed occupancy rate was approximately 50%
